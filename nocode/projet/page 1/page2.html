HTML/CSS : La base du web

Ce que j'ai aimé : Apprendre à structurer et styliser une page web de manière intuitive et créative.

Ce qui m'a paru difficile : Comprendre les concepts avancés comme le responsive design et l'utilisation des media queries.

Un projet marquant : La réalisation d’un site vitrine pour un faux restaurant. Cela m’a permis d’utiliser les flexbox, de créer un menu interactif et de rendre la page adaptable aux mobiles.

Algorithmie : La logique derrière le code

Ce que j'ai aimé : La satisfaction de résoudre des problèmes complexes avec des algorithmes bien pensés.

Ce qui m'a paru difficile : Travailler sur les récursions et comprendre la complexité algorithmique.

Un projet marquant :La realisation d'un trieur de nombres entré par l'utilisateur.

Javascript : Dynamiser les pages web

Ce que j'ai aimé : Ajouter de l’interactivité à une page, comme des formulaires dynamiques ou des animations.

Ce qui m'a paru difficile : Comprendre le fonctionnement du DOM et manipuler les événements de manière fluide.

Un projet marquant :La realisation d'un jeu snake. Il m'a permis de mieux comprendre l'utilisation de tout ce que j'ai vu pendant le mois et de retravailler les éléments que j'avaismal compris.
